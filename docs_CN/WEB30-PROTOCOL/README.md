# WEB30 Protocol

## 1. 目录定位

`WEB30-PROTOCOL` 是 NOVOVM 迁移中的应用/经济治理层文档区，不属于共识内核（`novovm-consensus`）本体。

按当前迁移口径，它主要对应：

- `F-10` Web3 存储服务
- `F-11` 域名系统
- `F-12` DeFi 核心
- `F-13` 多链插件能力
- 以及发币/治理规则在 NOVOVM 主链路的迁移说明
