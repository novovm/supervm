# SuperVM 主链 Token 经济模型 - 快速参考

## 🎯 一句话总结

**双轨收入(本币+外汇) → 80/20分红 → 每日主动领取 → 多资产储备背书 → 持续通缩 → 价值增长**

## 📊 核心数据

| 指标 | 数值 | 说明 |
|------|------|------|
| **M0 总量** | 100,000,000 | 固定上限,永不增发 |
| **分红比例** | 80% | 协议收入返还持币者 |
| **销毁率** | -3% ~ -5%/年 | 多维度销毁机制 |
| **年化分红** | >15% | 持币者收益目标 |
| **储备率** | >80% | 外汇储备健康指标 |
| **治理阈值** | 50-75% | 按提案类型分级 |

## 💰 货币层级

```
M0 (100M固定)
  ├─ 40% 矿工激励池 (10年线性释放)
  ├─ 20% 生态基金 (治理控制)
  ├─ 15% 团队预留 (2年锁定+3年线性)
  ├─ 10% 早期投资者 (1年锁定+2年线性)
  ├─ 10% 国库储备 (支付矿工)
  └─ 5% 流动性引导 (立即可用)

M1 (流通货币)
  = M0解锁 + 服务费再分配 - 销毁 - 锁仓

M2 (信用货币)
  = M1 × 150%抵押率 (稳定币/借贷)
```

## 💸 收入分配

### 本币收入(Token支付)

```
Gas费: 60%矿工 + 30%分红池 + 10%销毁
手续费: 50%分红池 + 20%基金会 + 20%矿工 + 10%销毁
```

### 外币收入(BTC/ETH/USDT支付)

```
外币 → 100%归集储备池 → 国库用Token等值支付矿工
矿工: 持有(分红) or 兑换(AMM)
```

## 🔥 销毁机制

| 类型 | 来源 | 年化影响 |
|------|------|---------|
| Gas BaseFee | 10-15% Gas费 | -2.0% |
| 手续费销毁 | 10% 手续费 | -0.5% |
| 清算罚金 | M2抵押不足 | -0.3% |
| 国库回购 | 治理决策 | -1.0% |
| **合计** | - | **-3.8%** |

## 🏛️ 治理结构

```
9人加权议会:
├─ 创始人 1席 (35%)
├─ Top5持币者 5席 (50%, 各10%)
├─ 团队代表 2席 (10%, 各5%)
└─ 独立委员 1席 (5%)

提案阈值:
├─ 参数调整: >50%
├─ 国库支出: >66%
├─ 协议升级: >75%
└─ 紧急冻结: >50% + 3类别席位
```

## 💎 分红领取

```
UTC 00:00 快照 → 计算可领取
用户调用 claim() → 转账分红
未领取自动累加 → 防止Gas浪费

公式: 可领取 = (用户持仓 / 总流通) × 当日分红池

限制:
- 最小持仓: 100 Token
- 领取频率: 每日一次
- Merkle Tree 证明优化
```

## 🔑 关键创新

1. ⭐ **储备替代机制**: 外币收入100%归集,Token支付矿工,类似数字央行
2. ⭐ **每日主动领取**: 避免自动分红Gas浪费,培养持有周期性
3. ⭐ **加权治理**: 平衡创始团队控制(35%)与社区去中心化(65%)
4. ⭐ **多资产背书**: BTC/ETH等外汇储备,增强Token信用
5. ⭐ **持续通缩**: 多维度销毁,供应持续减少推动价值增长
