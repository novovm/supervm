## 4. 与经济模型强关联的代码文件

> 这些文件虽然在 `contracts/web30/` 下,但与经济模型设计紧密关联,在阅读文档时可以对照参考。

- `contracts/web30/core/src/mainnet_token.rs`  
  主链 Token 接口定义,包含转账、销毁、分红相关的基础操作接口。

- `contracts/web30/core/src/treasury.rs`  
  国库模块,负责收入归集、外币储备、NAV 赎回、债券/稳定币相关资金流。

- `contracts/web30/core/src/foreign_payment.rs`  
  外币支付处理模块,实现“外币进入储备 + 用 Token 支付矿工”的替代逻辑。

- `contracts/web30/core/src/dividend_pool.rs`  
  分红池模块,实现每日快照与主动领取式分红。

- `contracts/web30/core/src/governance.rs`  
  治理模块,实现 9 人加权议会、提案、投票、执行等逻辑。

- (未来) 债券与 CDP 相关模块文件  
  待实际实现落地后,可以在此索引中补充对应路径。

## 5. 建议的阅读顺序

如果你是**从零重新进入项目**,想快速恢复对经济模型的整体理解,建议按以下顺序阅读:

1. `金融体系全景-总结.md`  → 建立整体概念
2. `经济模型与治理.md`      → 理解完整设计细节
3. `SuperVM-经济白皮书.md`   → 以对外话术再过一遍结构
4. `双轨定价机制-刚性兑付与市场交易.md` → 强化对 NAV 与双价机制的理解
5. `经济模型FAQ.md`          → 补齐关键疑问与边界条件
6. `ROADMAP.md`              → 对照看经济模块的实际落地节奏
7. `TOKEN-IMPLEMENTATION.md` / 相关 Rust 源码 → 进入开发/审阅视角
